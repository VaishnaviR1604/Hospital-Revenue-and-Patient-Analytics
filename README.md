# 🏥 Hospital Revenue and Patient Analysis Dashboard

## 📌 Project Overview

This is an end-to-end Data Analyst project focused on analyzing hospital data to generate actionable insights related to patient trends and revenue performance. The project covers the complete data analytics lifecycle including ETL process, data preprocessing, SQL-based analysis, and interactive dashboard development using Power BI.

---

## 🎯 Aim

To analyze hospital data and transform raw data into meaningful insights for better decision making.

---

## 📊 Objectives

* Identify key performance indicators (KPIs) such as total patients and total revenue
* Analyze patient-wise billing and treatment trends
* Perform insurance provider and demographic analysis
* Build an interactive dashboard for business insights

---

## 🛠️ Tech Stack

* Excel – Data Cleaning & Preprocessing
* SQL Server – Data Storage & Analysis
* Power BI – Dashboard & Visualization
* DAX – Calculated Measures

---

## 🔄 ETL Process

* **Extract**: Collected raw dataset from Kaggle
* **Transform**: Cleaned and preprocessed data using Excel
* **Load**: Stored cleaned data into SQL Server

---

## 📂 Data Source

* Dataset collected from Kaggle
* Includes patient details, billing information, treatment data, and insurance provider details

---

## 🧹 Data Preprocessing

* Handled missing values
* Removed duplicate records
* Standardized column formats
* Ensured data accuracy and consistency

---

## 🗄️ Data Storage

* Used SQL Server for storing structured data
* Created tables and maintained relationships

---

## 🔍 Data Analysis (SQL)

* Used SQL Joins to combine multiple tables
* Used Aggregate Functions (SUM, COUNT, AVG)
* Performed analysis to answer key business questions:

  * Total Patients
  * Total Revenue
  * Patient-wise Billing

---

## 📊 Data Visualization (Power BI)

* Imported data into Power BI
* Performed Data Modeling (table relationships)
* Created DAX measures for KPIs
* Built an interactive and user-friendly dashboard

---

## 📈 Dashboard Features

* KPI Cards (Total Revenue, Total Patients)
* Revenue Trend by Month (Line Chart)
* Treatment-wise Revenue (Column Chart)
* Insurance Provider Analysis (Bar Chart)
* Patient Gender Distribution (Pie Chart)
* Slicers (Month, Payment Method)

---

## 💡 Key Insights

* Identified monthly revenue trends and patterns
* Highlighted high-performing treatments
* Compared insurance provider contributions
* Analyzed patient demographics

---

## 🔁 Workflow

Excel ➝ SQL Server ➝ Power BI

---

## 🚀 Conclusion

This project demonstrates how raw hospital data can be transformed into actionable insights using an end-to-end data analytics workflow. The dashboard enables stakeholders to monitor performance, analyze trends, and make data-driven decisions.
