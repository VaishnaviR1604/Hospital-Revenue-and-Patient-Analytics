🏥 Hospital Revenue and Patient Analytics Dashboard
📌 Project Overview

This is a real-world hospital analytics project. I took a dataset from Kaggle, cleaned it in Excel, stored it in SQL Server, and built an end-to-end Power BI dashboard using data modeling, Power Query, and DAX.

The project helps analyze hospital revenue, patient flow, and treatment performance.

🛠 Tools Used
Excel – Data Cleaning & Preprocessing
SQL Server – Data Storage & Querying
Power BI – Dashboard & Visualization
DAX – KPI Measures & Calculations
Power Query – Data Transformation
🔄 End-to-End Workflow
Downloaded hospital dataset from Kaggle
Cleaned missing values, formatted columns, and standardized data in Excel
Loaded the cleaned data into SQL Server
Created relationships and star schema in Power BI
Built DAX measures for KPIs
Designed dashboard visuals, slicers, and filters
📊 Dashboard KPIs & Visuals
Revenue KPI – Overall revenue performance
Revenue Trend – Monthly revenue changes
Treatment Trend – Number of treatments over time
Patient Gender Split – Male vs Female distribution
Insurance Provider Analysis – Top insurance companies
Payment Type Slicer – Filter by payment method
Month Slicer – Filter data by month


🧠 Key Insights
Identified monthly revenue growth trends
Found the most common treatment categories
Analyzed male vs female patient ratio
Listed top insurance providers
Studied payment type usage patterns
